Economics

Agent-based Simulation of Market Exchange Trading and Scenarios from Microeconomics

These projects are libraries that can be assembled to form an agent-based market simulator with robot traders.

  • market-engine
  • market-example-contingent
  • market-agents

An example simulator, supporting one market, built atop this code is at:

  • single-market-robot-simulator

With plugins for additional functionality:

  • single-market-robot-simulator-viz-plotly
  • single-market-robot-simulator-savezip

A web front end is at

  • robot-trading-webapp

which provides the example simulator app at http://drpaulbrewer.github.io/robot-trading-webapp

Miscellaneous

Javascript NPM Libraries

  • stepify-plotly Makes step charts on Plotly from line charts by scanning traces for .steps:1
  • cumulative-distribution-function Creates empirical cumulative distribution function from point data
  • positive-number-array Returns an array of positive numbers from string or array input
  • transpluck Transposes CSV-like data to collection of columns and plucks desired data columns
  • csv-file-creator Creates a .csv file on the browser from array-of-array data.
  • find-zero-range Finds a line segment where a step function is zero, by bisection.
  • partial-index Partial sorting of muti-property data

Prototypes

  • multicoresql

Sharding of sqlite database and multi core processing of sqlite queries across collections of sqlite shards.

Examples/Utilities/Forks

  • docker-stacks

Private fork of Jupyter project docker stacks to enable and test some disabled functionality.

  • nginx-reverse-proxy

Quick setup of nginx web server in reverse proxy mode for proxying an app. Also provides an example of autodeployment of a Docker container to Dockerhub

  • json-editor-customization-textarea-as-positive-number-array

Modifying JDorn's json-editor to use a textarea to prompt for a list of positive numbers
